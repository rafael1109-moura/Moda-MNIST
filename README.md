# Moda-MNIST

Visão Geral do Projeto
Este projeto demonstra o carregamento e a preparação do popular conjunto de dados Fashion MNIST para tarefas de classificação de imagens. O Fashion MNIST serve como um substituto moderno e mais desafiador do que o MNIST original (dígitos escritos à mão) para o desenvolvimento e teste de modelos de Visão Computacional, como Redes Neurais Convolucionais (CNNs).

O objetivo principal é classificar imagens de artigos de vestuário em 10 categorias diferentes.



Tecnologias Utilizadas

Python

TensorFlow

TensorFlow Datasets (TFDS): Utilizado para carregar e gerenciar o dataset de forma eficiente.
